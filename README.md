# Software Developer Portfolio

> My personal software development portfolio showcasing production applications, technical projects, and ongoing software engineering work.

A modern, responsive software developer portfolio built with React and Vite.

The site serves as a central hub for showcasing software projects, technical skills, certifications, and professional experience through a clean, premium interface focused on usability and performance.

---

## Live Website

🌐 **https://donwilson-dev.com**

Explore the live portfolio to view featured applications, project documentation, certifications, and ongoing software development work.

## Features

- Responsive desktop, tablet, and mobile layouts
- Premium dark-themed user interface
- Project showcase with dedicated application pages
- Interactive Home Hero App Grid
- Professional About section
- Skills and technology overview

- Contact page with GitHub and LinkedIn integration

## Getting Started

### Clone the repository

```bash
git clone https://github.com/donwilson-dev/Portfolio.git
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
